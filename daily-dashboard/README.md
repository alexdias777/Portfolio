## Daily Dashboard

Dashboard desenvolvido com **TypeScript**, reunindo funcionalidades comuns do dia a dia.

## Funcionalidades
- Clima via API
- Hora em tempo real
- Busca de CEP (ViaCEP)
- Lista de tarefas com localStorage

## Tecnologias
- HTML
- CSS
- TypeScript
- APIs públicas

## Objetivo
Projeto para estudo e portfólio front-end.
